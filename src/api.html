DropdownList Object

The DropdownList Object is used for show options drop-down list

Constructor

                                               
                                            
var dropdownlist = new UI.DropDownList(element,items);
Parameters:

HTMLElement element.
JSON Array items. 

                                                    
var items = [
    {option:1,value:'Value a'},
    {option:2,value:'Value b',selected:true},
    {option:100,value:'Value n'}
];

The option attribute can be a Number or String. To validations add option:-1, and the attribute data-name. You can see an example here.

Returns:

The function that created the Dropdownlist object.

Methods

selectItem(option:Any) 
                                                
dropdownlist.selectItem(option);

Set the drop-down list with the option.

Parameters:

Any option.

getOption():Any 
                                                
dropdownlist.getOption();

Get the selected option.

Returns:

Any option.

Get the selected value.

Returns:

Any value.

dropdownlist.getItem();

Get the selected item.

JSON item.

dropdownlist.getItemByOption(option);

Parameters:

Any option.

Returns:

JSON item.

Get item by value.

Form Object

The Form Object is used for :

  • Validate inputs of the form.
  • Detect whether there were changes in the form.
  • Get form data as JSON.
  • Get form data as serialized.
  • Send form data as JSON by Ajax.
  • Send form data as JSON by Web Sockets.
  • Send form data as serialized by Ajax
  • Send form data as serialized by Web Sockets
  • Upload files on sever.

Constructor

                                               
                                            
var form = new UI.Form(element);
Parameters:

HTMLElement form element.

Returns:

The function that created the Form object.

Methods

isValid():Boolean 
                                                
form.isValid()

Check the fields of form depending on the validations of each field, if all fields are correct this returns true.
Any input that needs to be validated is required to have the following structure: 

                                                
<div class="form-group">
 <div class="col-sm-12">
    <input type="text" name="user" class="form-control" data-required="true">
 </div>
</div>

Note that the data attribute is for define the field validation, you can have all validations that you need. List of built-in Validation methods here

Returns:

Boolean, true if all fields are correct.

hasChanged():Boolean 
                                                
form.hasChanged()

Check if changes have been made in the form inputs.

Returns:

Boolean, true if has been changed in the form inputs.

serialize():String 
                                                  
form.serialize()

Create a text String in standard URL-enconded notation.
Required name attribute in inputs and textarea, for working with the dropdownlist used the data-name attribute.

Returns:

String URL-encoded notation

toJSON():JSON 
                                                  
form.toJSON()

Create a JSON with the elements form.
Required name attribute in inputs and textarea, for working with the dropdownlist used the data-name attribute.

Returns:

JSON

Custom Controls


The form controls are Checkboxes and Radio Buttons custom, you only need to add a class name and the result is awesome,the classes are flexbles and can be added to Checkboxes like Radio Buttons. Is required use the following structure: 

                                                 
 <span class="ui-form-control ui-form-control-f">
  <input id="ui-form-control-f" type="checkbox">
  <label for="ui-form-control-f">
	ui-form-control-f
  </label>
 </span>
List styles classes
ui-form-control-f ui-form-control-o ui-form-control-toggle
ui-form-control-circle-o ui-form-control-circle-f ui-form-control-simple
ui-form-control-dat-circle-o ui-form-control-dat-circle-f ui-form-control-thumbs-up
ui-form-control-thumbs-down ui-form-control-line-through ui-form-control-square
ui-form-control-tool-bar : for this class is required use the following structure: 
				                                                
<div class="form-group ui-inline">                  
   <span class="ui-form-control ui-form-control-tool-bar">
    <input id="opt-1" name="opt" type="radio">
    <label for="opt-1" >
     Option 1
    </label>
   </span>                        
   <span class="ui-form-control ui-form-control-tool-bar">
    <input id="opt-2" name="opt" type="radio">
    <label for="opt-2" >
     Option 2
    </label>
   </span>     
</div>
* You can display group using ui-inline or ui-block classes.
Large classes
ui-1x ui-2x ui-3x ui-4x ui-5x
Example: 
				                                              
 <span class="ui-form-control ui-form-control-f ui-3x">
  <input id="ui-form-control-f" type="checkbox">
  <label for="ui-form-control-f">
	ui-form-control-f
  </label>
 </span>
Color classes
ui-primary ui-success ui-info ui-warning ui-danger
Example: 
				                                              
 <span class="ui-form-control ui-form-control-f ui-primary">
  <input id="ui-form-control-f" type="checkbox">
  <label for="ui-form-control-f">
	ui-form-control-f
  </label>
 </span>
