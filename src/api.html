Notify Object

The Notify Object is used for show messages

Constructor

                                               
                                                  
var notify = new UI.Notify(HtmlElement);
Parameters:

HtmlElement for display the notification.

Returns:

The function that created the Notify object's.

Methods

success(message:String) 
                                                  
notify.success('Success Notify');

Display the notification the type Success.

Parameters:

String message for display in the notification.

info(message:String) 
                                                  
notify.info('Info Notify');

Display the notification the type Info.

Parameters:

String message for display in the notification.

warning(message:String) 
                                                  
notify.warning('Warning Notify');

Display the notification the type Warning.

Parameters:

String message for display in the notification.

danger(message:String) 
                                                  
notify.danger('Danger Notify');

Display the notification the type Danger.

Parameters:

String message for display in the notification.

wait(message:String) 
                                                  
notify.wait('Wait Notify');

Display the notification the type Wait.

Parameters:

String message for display in the notification.

close() 
                                                  
notify.close();

Close the notification.

DropdownList Object

The dropdownList Object is used for show option list

Constructor

                                               
                                             
var dropdownlist = new UI.DropDownList(HtmlElement, Items[]);
Parameters:

HtmlElement for display the dropdown list.
Items is Json array as this: 

                                                       
var items = [
              {option:1,value:'Value a'},
              {option:2,value:'Value b'},
              {option:'n',value:'Value n'}
];

Returns:

The function that created the Dropdownlist object's.

Methods

init(option:Any) 
                                                  
dropdownlist.init(option);

Initialize the object with the option selected.

Parameters:

Option of the item selected.

getOption():Any 
                                                
dropdownlist.getOption();

Get the option selected.

Returns:

Option of the item selected.
